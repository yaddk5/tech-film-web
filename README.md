# tech-film-web
This website is a digital film diary for tech films inspired by Letterboxd. It includes an about section, as well as sections for favorite tech films, watch list and reviews.

## Technologies
- HTML
- CSS

## Features
- Movie cards
- Watch trailer links
- Custom styling

## Design process
The project began as a hand-drawn prototype before implementation. The prototype is in included in the 'sketch' folder to demonstrate the planning and design process.
